## 移动案例

包括京东和苏宁两个移动端的界面案例

1. ### 苏宁商城移动端项目

   + 通过rem伸缩布局，能伸缩适配十余种主流屏幕的宽度
   + 使用zepto.js完成页面的动效
   + 通过手写手势封装，达到轮播图的自动轮播和手势切换自动吸附等效果

2. ### 京东移动商城练习

   + 界面百分比布局
   + 页面： 首页 index.html , 分类页 category.html
   + 基于原生JS
   + 个人练习原生JS的小项目，仿京东商城页面效果，顶部搜索栏随滑动变色
   + 轮播图封装，手势切换及吸附效果。
   + 分类页面使用iscroll.js插件实现区域滑动。
